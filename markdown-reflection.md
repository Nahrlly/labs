# Markdown Reflection

## What I learned

- I learned that Markdown uses simple symbols to format text.
- I learned that headings help organize ideas.
- I learned that lists make notes easier to scan.

## What I tried

- I tried writing headings with `#` symbols.
- I tried making a bullet list.
- I tried adding blank lines between sections.

## Evidence or examples

Here is one Markdown example I can use again:

```markdown
## My Heading

- First idea
- Second idea
- Third idea
```

## What confused me

- I was confused by where blank lines are needed.
- I was confused by how preview mode changes the way Markdown looks.
- I was confused by when to use one `#` or two `##` symbols.

## Preview fix

- I will use preview mode to check my Markdown.
- I will look for missing blank lines.
- I will make sure each list item is on its own line.

## Next steps

- Practice writing more Markdown notes.
- Compare the source view with the preview view.
- Fix one formatting mistake at a time.
